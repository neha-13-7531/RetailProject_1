Retail Sales Analytics - Power BI Dashboard

This folder contains the Power BI visualization layer of the project.

Dashboard Pages

1. Executive Overview
High-level business KPIs, including:
- Total Revenue
- Total Orders
- Customer Count
- Sales Trends

2. RFM Segmentation
Customer segmentation based on:
- Recency
- Frequency
- Monetary value

3. Cohort Analysis
Customer retention analysis by first purchase month.

4. Data Model
Star schema structure connecting fact and dimension tables.
