Retail Sales Analysis - SQL Layer

This folder contains SQL scripts used for the data preparation
and analysis stages of the Retail Analytics project.

SQL Pipeline:

1. Database Setup
Creates tables and loads raw data.

2. Data Cleaning
Handles missing values and incorrect formats.

3. Star Schema
Creates fact and dimension tables for analytics.

4. RFM Analysis
Calculates Recency, Frequency, Monetary scores.

5. Cohort Analysis
Analyzes customer retention patterns.

6. Business Queries
Generates insights used in the Power BI dashboard.
