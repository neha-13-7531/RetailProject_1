Retail Sales Analysis - Python Layer

This folder contains Python scripts used for advanced analytics.

Files Included:

1. data_loader.py
Connects Python to the PostgreSQL database.

2. rfm_segmentation.py
Calculates Recency, Frequency, and Monetary scores for customers.

3. market_basket_analysis.py
Performs association rule mining to identify products that are frequently purchased together.

4. rfm_customer_analysis.ipynb
Exploratory notebook for customer behavior analysis.
